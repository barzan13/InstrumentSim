# InstrumentSim Project Plan

## Immediate Goal

Build a functional prototype proving:

Input -> Audio -> Visual Feedback

## Early Architecture Goals

- Separate audio logic from UI logic
- Create reusable instrument models
- Support future MIDI expansion
- Keep rendering responsive

## Prototype Instrument

Initial instrument:

- Piano keyboard

Reasoning:

- Familiar layout
- Easy note mapping
- Clear visual feedback
- Excellent for testing input/audio systems

## Future Instrument Ideas

- Guitar fretboard
- Saxophone key map
- Drum pads
- Synthesizer interface
- Violin fingerboard

## Technical Areas To Explore

### Audio

Potential libraries:

- NAudio
- CSCore
- FMOD

### UI

Potential visuals:

- Animated key highlights
- Glow effects
- Reactive instrument parts
- Performance overlays

### Recording

Potential systems:

- MIDI event recording
- WAV export
- Timeline playback
- Session management

## Long-Term Vision

Eventually evolve InstrumentSim into a hybrid between:

- Practice tool
- Instrument visualizer
- MIDI playground
- Creative audio sandbox
